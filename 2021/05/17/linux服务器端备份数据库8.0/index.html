linux服务器端备份数据库8.0

ssh远程本地连接服务器

ssh root@47.103.202.240

格式：ssh 用户名@服务器ip

利用mysqldump语句对指定表进行备份

mysqldump -uroot -pjsBu123456 flag students >../www/web/a.sql;

mysqldump -u用户名 -p密码 库名 表名 > ../地址路径/文件名

显示这个即为成功

[Warning] Using a password on the command line interface can be insecure.

本地数据库命令行备份

mysqldump -uroot -pjsBu123456 flag students >D:/a.sql

mysqldump -u用户名 -p密码 库名 表名 > ../地址路径:/文件名

显示这个即为成功

mysqldump: [Warning] Using a password on the command line interface can be insecure.
