scp协议

1.从服务器复制文件到本地：

root@192.168.1.100 root是目标服务器（有你需要拷贝文件的服务器）的用户名，192.168.1.100是IP地址，后面紧跟的 “：” 不要忘记，/data/test.txt（多文件还有test1.txt，test2.cpp，test3.bin，test.a，test.c等） 是目标服务器中你要拷贝文件的地址，接一个空格，后面的 /home/myfile/ 是本地接收文件的地址。

2.从服务器复制文件夹到本地：

只需在前面加 -r 即可，就可以拷贝整个文件夹。

3.从本地复制文件到服务器：

4.从本地复制文件夹到服务器：

本地文件上传至服务器

scp D:/桌面/Flag/Flag/target/flag-0.0.1-SNAPSHOT.jar root@47.103.202.240:/www/web

scp -r D:/下载/dist/* root@47.103.202.240:/yiban/pages/211/ //上传文件夹里所有文件

scp D:/请注意签收.zip root@47.103.202.240:/yjxphoto

scp 本地路径 服务器用户名@服务器ip:/在服务器存放的目录

从服务器传输文件到本地

scp root@47.103.202.240:/www/web/a.sql D:/桌面

scp -r root@192.168.1.100:/data/ /home/myfile/

服务器jar程序的启动与进程的查看与结束

在服务器启动jar包程序

1、先进入jar包存放位置 cd /目录/目录

2、java -jar jar包名称 (启动后 ctrl + c 结束 接下一步)

​ java -jar flag-0.0.1-SNAPSHOT.jar

3、nohup java -jar jar包名称 > nohup.out &

nohup java -jar flag-0.0.1-SNAPSHOT.jar > nohup.out &

进程的查看与结束

1、查看所有进程

​ ps aux

2、结束进程

​ kill -9 进程ID

3、抓取日志

​ cat nohup.out

或

​ vim nohup.out
