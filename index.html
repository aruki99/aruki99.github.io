title:HTML5

简介

超文本标记语言（英语：HyperText Markup Language，简称：HTML5）是一种用于创建网页的标准标记语言。

​ 您可以使用 HTML 来建立自己的 WEB 站点，HTML 运行在浏览器上，由浏览器来解析。

HTML 是用来描述网页的一种语言。

  • HTML 指的是超文本标记语言: HyperText Markup Language
  • HTML 不是一种编程语言，而是一种标记语言
  • 标记语言是一套标记标签 (markup tag)
  • HTML 使用标记标签来描述网页
  • HTML 文档包含了HTML 标签及文本内容
  • HTML文档也叫做 web 页面

标签

标签由<>包裹着，并且成对出现。

第一个标签为“开始标签”（开放标签）

第二个标签为“结束标签”（闭合标签）

特殊字符

&nbsp 空格

&gt 大于符号

&lt 小于符号

&copy 版权所有符号
